Tipe Data yang ada di Javascript

Javascript merupakan bahasa pemrograman yang digunakan untuk membuat tampilan website agar tampak dinamis, memberikan kemampuan kepada pengembang web untuk membuat efek dinamis, menangani interaksi pengguna, dan mengubah konten halaman secara real-time

JavaScript dapat dituliskan ke dalam html seperti berikut ini langsung didalam satu file HTML

                    !DOCTYPE html
                    html lang="en"
                    head
                        title>Test Javascript/title
                    /head
                    body
                        script
                            document.write(5+6);
                        /script
                    /body
                    /html

Berikut adalah cara mengoutputkan hasil dari 2 buah variabel, cara ini juga berlaku pada variabel tipe data string

                    !DOCTYPE html
                    html lang="en"
                    head
                        title>Test Javascript/title
                    /head
                    body
                        script
                            document.write(5+6);
                        /script
                    /body
                    /html

Berikut merupakan contoh penggunaan javascript untuk menampilkan ouput!

                    alert("Saya sedang belajar JavaScript!");
                    let x = 2;
                    let y = 10;
                    const hasil = x * y;
